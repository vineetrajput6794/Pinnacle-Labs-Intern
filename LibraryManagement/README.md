# 📚 Library Management System

A simple console-based Java application to manage a small library's inventory. It allows users to perform basic operations like viewing available books, borrowing, returning, adding new books, and searching for books.

---

## ✨ Features

- **View Inventory:** View a list of all books available in the library with their status.
- **Borrow Book:** Borrow a book by entering its unique ID. Ensures that already borrowed books cannot be borrowed again.
- **Return Book:** Return a borrowed book using its ID.
- **Add Book:** Add new books to the library inventory by specifying ID, Title, and Author.
- **Search Book:** Find specific details of a book using its ID.

---

## 🛠️ Technologies Used

- **Language:** Java (Core)
- **Concepts:** Object-Oriented Programming (Classes, Objects, Methods), Collections (`ArrayList`), User Input (`Scanner`)

---

## 🚀 How to Run

1. Open the project in any Java IDE like IntelliJ IDEA, Eclipse, or VS Code.
2. Locate the `LibraryManagementSystem.java` file in the `src` folder.
3. Run the `main` method in `LibraryManagementSystem`.
4. Follow the interactive console menu to manage the library system!
