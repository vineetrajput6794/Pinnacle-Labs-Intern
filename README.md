# Weather App

## About the Project
This is a web-based Weather Application developed in Java. It allows users to view weather information using a dynamic and responsive web interface. The application fetches weather data and presents it to the user through a clean UI, leveraging Java Servlets and Jakarta EE for the backend.

## Key Features
- **Real-Time Weather Data:** Displays current weather information.
- **Dynamic Web Pages:** Utilizes JSP (JavaServer Pages) and HTML for a dynamic frontend.
- **RESTful Integration:** Parses JSON data to provide accurate and updated weather information.
- **Clean User Interface:** Styled with CSS for an intuitive and responsive layout.

## Technologies Used
- **Backend:** Java 17, Jakarta Servlet API
- **Frontend:** HTML5, CSS, JSP
- **Data Parsing:** `org.json` library for processing JSON data
- **Build Tool:** Maven (pom.xml)
- **Deployment:** Packaged as a WAR file for deployment on web servers like Apache Tomcat.
