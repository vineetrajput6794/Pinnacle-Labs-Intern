# Pinnacle Labs - Java Programming Internship

Welcome to the repository for my Java Programming Internship at Pinnacle Labs! This repository serves as a centralized portfolio for all the projects, assignments, and tasks I have completed during my internship.

## About the Internship
During this internship, I have been focused on developing robust, scalable, and efficient applications using Java and modern web technologies. The experience has provided me with hands-on exposure to full-stack development, server-side programming, and software engineering best practices.

## Key Features of the Repository
- **Organized Structure:** Each project is contained within its own dedicated subdirectory (e.g., `WeatherApplication/`) with its own source code, configurations, and project-specific README.
- **Progressive Learning:** The projects demonstrate a progression of skills, starting from core Java concepts to building dynamic, RESTful web applications.
- **Best Practices:** Focus on clean code, proper project structure, and standardized build processes.

## What I've Learned
- **Java Web Development:** Building dynamic web applications using Jakarta EE, Java Servlets, and JSP.
- **Data Handling:** Fetching, parsing, and processing JSON data effectively.
- **Frontend Integration:** Connecting backend Java services with frontend HTML/CSS to create clean and responsive user interfaces.
- **Version Control:** Managing project versions and collaborating using Git and GitHub.
- **Project Organization:** Structuring enterprise-level applications correctly.

## Tools & Technologies Used
- **Languages:** Java (JDK 17), HTML5, CSS, JavaScript
- **IDEs & Editors:** IntelliJ IDEA (Primary IDE for Java development)
- **Build Tools:** Maven (Dependency management and project building)
- **Server:** Apache Tomcat / SmartTomcat
- **Version Control:** Git & GitHub
